Bienvenido a la plantilla del editor, aquí puedes ser libre de editar cualquier texto o imagen del iframe (página dentro de otra) . Por si quieres editar el código, el boton "Ver código" te permitirá ver y editar el código fuente HTML de la página.

Al lado de el boton anteriormente explicado están los cuadros de texto correspondientes al id y class del elemento html (también el del color del texto pero lo obviaremos) , si has tenido contacto con HTML sabrás para qué son estás propiedades, sino te lo explicamos a continuación.

- Propiedad id

indentificador el cual se lee normalmente con CSS con selectores para añadirle estilizados y también con JavaScript para añadirle funcionalidades, cabe destacar que es recomendable no poner un mismo id a dos elementos por lo cual se usan en elementos especiales.

- Propiedad class

Al igual que el anterior es un indentificador el cual se lee normalmente con CSS con selectores para añadirle estilizados y también con JavaScript para añadirle funcionalidades a un elemento html, pero a diferencia del id este sí se puede usar en diferentes elementos html.
